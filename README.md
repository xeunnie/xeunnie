## Core Principles
* **Architecture-Driven Development:** 확장 가능한 아키텍처를 설계하며, 코드의 유지보수성과 재사용성을 최우선으로 고려합니다.
* **Reasoning over Implementation:** 기술적 의사결정의 근거를 중요하게 생각하며, '어떻게'보다 '왜'에 집중하여 최적의 솔루션을 도출합니다.
* **Full-stack Perspective:** 백엔드 시스템 및 인프라에 대한 높은 이해도를 바탕으로, 프론트엔드와 백엔드 사이의 유연한 데이터 흐름을 설계합니다.
